# Contributing

Thank you for your interest in improving one-click-portfolio! 

## How to submit a new feature
1. Fork the repository
2. Create a feature branch
3. Keep changes minimal (remember: zero build tools, pure HTML/CSS/JS)
4. Verify your changes do not break the design for existing fields
5. Open a Pull Request

## Reporting Bugs
If a particular data combination breaks the layout or causes a crash, please open an Issue with your `data.json` snippet.

## Pull Request Checklist
- [ ] No build tools added
- [ ] No external CDN links added (keep it dependency free)
- [ ] Accessibility maintained (ARIA labels, good contrast)
- [ ] `validate.js` passes on default template
